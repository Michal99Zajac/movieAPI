# movieAPI

## FastAPI for movies

Part of Natural Language Processing subject on University of Technology in Cracow

## run server

##### install pipenv

```bash
pip install pipenv
```

##### install packages

```bash
pipenv install
```

##### run shell

```bash
pipenv shell
```

##### run server

```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```
